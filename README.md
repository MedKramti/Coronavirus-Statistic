# Coronavirus-Statistic

A simple script to scrap data from corona.help.

----

## An Example 

```java
Scrapper scrapper = new Scrapper();
scrapper.globalStats("tunisia"); // Change tunisia to any country
```
_Expected output:_

* Confirmed Cases
* Total Death
* Total Recovered
* Total Active Cases
